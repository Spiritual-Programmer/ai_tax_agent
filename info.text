1. Employee Information (Your info)
Employee Name – First, last, and middle initial.

Employee Address – Street, city, state, ZIP.

Social Security Number (SSN) – 9-digit SSN.

2. Employer Information

Employer Name – Company name.

Employer Address – Street, city, state, ZIP.

Employer Identification Number (EIN) – 9-digit federal tax ID.

Control Number – Optional internal number for employer records.

3. Wage and Tax Information

Box 1: Wages, tips, other compensation – Total taxable wages.

Box 2: Federal income tax withheld – Total federal tax withheld.

Box 3: Social security wages – Wages subject to Social Security tax.

Box 4: Social security tax withheld – Amount withheld for Social Security.

Box 5: Medicare wages and tips – Wages subject to Medicare tax.

Box 6: Medicare tax withheld – Amount withheld for Medicare.

Box 7: Social security tips – Tips reported subject to Social Security.

Box 8: Allocated tips – Tips allocated by employer.

Box 10: Dependent care benefits – Pre-tax dependent care contributions.

Box 11: Nonqualified plans – Contributions to nonqualified retirement plans.

Box 12 – Various codes for benefits, e.g., 401(k), adoption assistance, etc.

12a, 12b, 12c, 12d – Each has a code and amount.

Box 13 – Checkboxes for:

Statutory employee

Retirement plan

Third-party sick pay

Box 14: Other – Employer-specific info (union dues, state disability, etc.)

4. State and Local Tax Information

Box 15: State / Employer’s state ID number

Box 16: State wages, tips, etc.

Box 17: State income tax

Box 18: Local wages, tips, etc.

Box 19: Local income tax

Box 20: Locality name – City or local jurisdiction
